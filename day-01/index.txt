1. HTML

1.1 HTML 기본구조
HTML(HyperText Markup Language)은 웹 페이지의 뼈대를 만드는 언어입니다.
쉽게 말하면, 웹 페이지라는 집을 짓는 설계도와 같습니다.
HTML 문서는 특정한 구조를 가지고 있으며, 브라우저는 이 구조를 읽고 화면에 웹 페이지를 보여줍니다.

가장 기본적인 HTML 문서는 다음과 같은 구조를 가집니다.

1.1.1 <!DOCTYPE html>
- HTML 문서가 HTML5 표준을 사용하고 있음을 브라우저에 알려주는 선언문입니다.
- 브라우저가 문서를 해석할 때, "이건 HTML5 방식으로 읽어야 한다."라고 이해하게 해줍니다.
- HTML 문서에서는 항상 맨 위에 위치해야 합니다.

1.1.2 <html lang="en">
- HTML 문서의 루트(root) 요소입니다.
- <html> 안에 문서의 모든 내용이 들어갑니다.
- lang="en" 속성은 문서의 주 언어가 영어임을 나타냅니다. (검색 엔진, 스크린 리더(시각장애인을 위한 화면 읽기 프로그램) 등)에서 활용됩니다.

1.1.3 <head>
- 문서의 메타 정보(정보를 담는 공간)를 정의하는 영역입니다.
- 브라우저에서 보이는 내용은 없고, 제목 & 문자 인코딩 & 스타일 & 외부 파일 링크 등을 담습니다.

1.1.4 <meta name="viewport" content="width=device-width, initial-scale=1.0" />
- 반응형 웹에서 매우 중요한 설정입니다.
- 모바일 기기에서 화면 크기를 맞추기 위해 사용하며, width=device-width는 화면 폭을 기기 화면 폭에 맞춥니다.
- initial=1.0은 초기 확대/축소 비율을 100%로 설정합니다.

뷰포트(Viewport)는 사용자가 보고 있는 웹 페이지의 화면 영역을 말합니다.
데스크톱 브라우저에서는 보통 브라우저가 창 크기가 뷰포트가 되고, 모바일 기기에서는 기기 화면 크기가 뷰포트가 됩니다.
initial-scale은 사용자가 페이지를 처음 열었을 때, 얼마나 확대된 상태로 보여줄지를 정하는 거예요.

1.1.5 <title>
- 브라우저 탭에 표시되는 페이지 제목을 정의합니다.

1.1.6 <body>
- 실제 화면에 보여지는 내용을 작성하는 영역입니다.
- 텍스트, 이미지, 링크, 버튼, 표 등 모든 시각적 요소가 이 안에 들어갑니다.
